# QRacer 摹码

<img src="assets/logo.svg" width="240" alt="logo"/>



QRacer 是一个面向柔印印前工作的 Windows 桌面工具，用于把二维码截图、拍照图或屏幕局部截图还原成可用于制版的 SVG 矢量文件。

它的目标不是“能扫码就行”，而是尽量让码点位置与原图保持 1:1 几何一致。应用会在导入后自动识别、校正、采样并生成预览，用户可以通过左右对比和差异高亮检查结果，再导出 SVG 或复制 SVG 文本到剪贴板后粘贴进 Illustrator。

## 🌟功能特性

- 支持从剪贴板粘贴图片、打开本地图片、框选截屏导入。
- 自动识别二维码、Data Matrix 码、微信小程序码和抖音码。
- 支持手动指定码类型，避免自动识别误判。
- 支持左右对比、差异高亮和状态栏差异数量提示。
- 支持导出 SVG 文件，也支持直接复制 SVG 文本到剪贴板。
- QR、Data Matrix、小程序码和抖音码均提供手动校准入口，用于校正不稳定的拍照图。
- 纯 Rust 实现，不依赖 OpenCV，目标是轻量单文件分发。

## 🐴支持的码类型

| 码类型 | 处理策略 | 说明 |
| --- | --- | --- |
| 二维码 (QR) | 解码重生成 + 8 掩膜对比；失败时网格像素匹配 | 支持标准、微信、小红书等外观输出 |
| Data Matrix 码 | ECC 200 网格采样 | 支持方形和矩形符号，不做 payload 重生成 |
| 微信小程序码 | 径向网格采样 | 私有协议，仅做几何还原 |
| 抖音码 | 同心圆网格采样 | 私有协议，仅做几何还原 |

## 🛖环境要求

- Windows。
- Rust 2024 edition 对应的稳定工具链。

安装 Rust 后，在项目根目录运行：

```powershell
cargo run
```

发布构建：

```powershell
cargo build --release
```

生成的可执行文件位于：

```text
target/release/qracer.exe
```

## 🚀快速使用

1. 启动程序后，用以下任一方式导入图片：
   - 点击 `粘贴 (Ctrl+V)` 或直接按 `Ctrl+V` 粘贴剪贴板图片。
   - 点击 `打开...` 选择 PNG/JPG/BMP/WebP 等图片。
   - 点击 `截屏`，框选屏幕中的码图区域。

2. 等待自动处理完成。左侧显示原图，右侧显示矢量化结果或差异预览，状态栏会显示识别结果、模块/像素差异等信息。

3. 如识别错误，可在工具栏 `码类型` 下拉框中改为手动类型，然后重新处理当前图片。

4. 检查结果：
   - QR 可切换 0-7 掩膜、自动选最佳，或使用 `网格像素匹配`。
   - Data Matrix、小程序码、抖音码可勾选 `显示差异`。
   - 差异颜色含义：红色表示原图有而生成图没有，蓝色表示原图没有而生成图有。

5. 如果拍照图自动校正不准，点击 `手动校准`：
   - 拖动整图移动。
   - 拖动边框或边中点小方块做水平/垂直缩放。
   - 拖动四角做缩放；按住 `Shift` 拖边框或四角可等比例缩放。
   - 按住 `Ctrl` 拖动四角才做自由透视变换。
   - 用滚轮微调缩放。
   - 用旋转手柄或 `逆时针 1°` / `顺时针 1°` 微调。
   - QR 可在弹窗内选择版本；Data Matrix 可选择 ECC 200 符号尺寸；抖音码可选择无框版/黑框版参考线。
   - 可用 `Ctrl+Z` 或 `撤销` 回退。
   - 对齐参考线后点击 `应用校准` 重新采样。

6. 确认结果后：
   - 点击 `导出 SVG` 保存文件。
   - 或点击 `复制到剪贴板`，再到 Illustrator 中粘贴为矢量对象。

## 🕸️项目结构

```text
src/
├── app.rs                  应用状态、后台处理、导入/导出流程
├── image_io.rs             剪贴板读图、文件打开、纹理转换
├── screen_capture.rs       Windows 框选截屏
├── detect/                 码类型识别和定位点检测
├── pipeline/               预处理、透视校正、网格推算
├── codec/                  QR/Data Matrix/小程序码/抖音码采样与生成
├── vector/                 SVG 输出和差异预览
└── ui/                     egui 界面组件
```

更多实现细节见：

- `docs/ARCHITECTURE.md`：架构、数据流、算法和技术决策。
- `docs/IMPLEMENTATION_PLAN.md`：阶段任务、验收标准和路线图。

## ⚠️注意事项

- 本项目当前仅面向 Windows。
- 小程序码和抖音码协议未公开，工具只做几何采样还原，不做私有 payload 逆向。
- Data Matrix 当前按 ECC 200 合法网格采样，不解码后重生成。
- SVG 用于印前生产前，请务必用差异高亮、扫码和人工叠图检查确认。
- 软件开源免费，许可证见 `LICENSE`。
